# Proyecto_Airdraw
Aquí tienes un párrafo aún más corto, **muy por debajo de 350 palabras** (son unas 40):  **AirDraw Secure es un sistema de visión por ordenador que utiliza una cámara para realizar autenticación mediante patrones visuales y, tras desbloquearse, permite seguir la mano del usuario para dibujar en el aire en tiempo real.
